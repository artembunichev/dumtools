dumtools - dummy devtools.

Google Chrome devtools dummier extension.


What Changes:

- Remove HTML, Editor and Console syntax highlighting.
- Remove bracket matching ( styles ).
- Remove console eager preview ( styles ).
- Remove autocompletion ( styles ).
- Remove argument hints ( styles ).
- Remove editor line folding ( styles ).
- Remove editor toolbar.
- Change most of text and icon colors to black.
- Change font and tab size.



Installation:

1. Clone this repo:
	git clone https://github.com/artembunichev/dumtools.git

2. Change styles, if you want ( e.g. font family or tab size ).

3. Open devtools, go to Settings -> Experiments and check "Allow extensions to load custom stylesheets".

4. Open "chrome://extensions/" turn on "Developer mode", click "Load unpacked" and select directory with dumtools extension.
